# odin-recipes# Odin Recipes 

This project is part of **The Odin Project** curriculum. The goal is to build a basic recipe website to practice fundamental HTML skills and working with the Git version control system.

## About the Project
The website consists of a main index page with links to individual recipes. Each recipe includes a list of ingredients and step-by-step instructions.

## Demonstrated Skills
In this project, I have practiced and demonstrated:
*   Building a basic **HTML structure** (boilerplate).
*   Working with **headings** (`<h1>` to `<h3>`) and **paragraphs** (`<p>`).
*   Creating **unordered** (`<ul>`) and **ordered** (`<ol>`) lists.
*   Linking pages using **relative paths** (`<a>`).
*   Embedding **images** (`<img>`) with alternative text.
*   Basic **Git workflow** (add, commit, push).

## How to Run
Simply open the `index.html` file in any modern web browser.
